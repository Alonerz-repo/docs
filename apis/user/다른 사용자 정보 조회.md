# Description

다른 사용자 정보 조회 API

# Request

## Method

```
GET /api/users/:userId
```

## Params

```json
{
  "userId": 0
}
```

# Response

## Success

### Status

```
200 Ok
```

### Data

```json
{
  "user": {
    "userId": 0,
    "kakaoId": null,
    "nickname": "",
    "thumbnail_image_url": null,
    "age": 0,
    "gender": "",
    "point": 0,
    "carrers": [
      {
        "careerId": 0,
        "part": "",
        "year": "",
        "description": ""
      }
    ]
  }
}
```

## Error

### Status

```
401 Unauthorized
403 Forbidden
404 Not Found
```

### Data

```json
{
  "statusCode": 0,
  "message": [""],
  "error": ""
}
```
