# Description

내 정보 수정 API

- 닉네임
- 나이
- 성별

# Request

## Method

```
POST /api/users/me
```

## Header

```json
{
  "Authorization": "Bearer ${ accessToken }"
}
```

## Data

```json
{
  "nickname": "",
  "age": 0,
  "gender": ""
}
```

# Response

## Success

### Status

```
200 Ok
```

## Error

### Status

```
400 Bad Request
401 Unauthorized
403 Forbidden
404 Not Found
```

### Data

```json
{
  "statusCode": 0,
  "message": [""],
  "error": ""
}
```
