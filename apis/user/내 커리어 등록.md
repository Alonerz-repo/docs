# Description

내 커리어 등록 API

# Request

## Method

```
POST /api/users/career
```

## Header

```json
{
  "Authorization": "Bearer ${ accessToken }"
}
```

## Data

```json
{
  "part": "",
  "year": 0,
  "description": ""
}
```

# Response

## Success

### Status

```
201 Created
```

## Error

### Status

```
400 Bad Request
401 Unauthorized
429 Too Many Request
```

### Data

```json
{
  "statusCode": 0,
  "message": [""],
  "error": ""
}
```
