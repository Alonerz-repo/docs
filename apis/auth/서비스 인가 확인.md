# Description

서비스 인가 확인 API

# Request

## Method

```
GET /api/auth
```

## Header

```json
{
  "Authorization": "Bearer ${ accessToken }"
}
```

# Response

## Success

### Status

```
200 Ok
```

### Data

```json
{
  "userId": 0,
  "kakaoId": null,
  "nickname": ""
}
```

## Error

### Status

```
401 Unauthorized
403 Forbidden
404 Not Found
```

### Data

```json
{
  "statusCode": 0,
  "message": [""],
  "error": ""
}
```
