# Description

서비스 회원가입 API

# Request

## Method

```
POST /api/auth/signup
```

## Body

```json
{
  "email": "",
  "nickname": "",
  "password": ""
}
```

# Response

## Success

### Status

```
201 Created
```

### Data

```json
{
  "accessToken": "",
  "refreshToken": ""
}
```

## Error

### Status

```
400 Bad Request
409 Conflict
```

### Data

```json
{
  "statusCode": 0,
  "message": [""],
  "error": ""
}
```
