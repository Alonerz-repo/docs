# Description

서비스 로그인 API

# Request

## Method

```
POST /api/auth/login
```

## Body

```json
{
  "email": "",
  "password": ""
}
```

# Response

## Success

### Status

```
200 Ok
```

### Data

```json
{
  "accessToken": "",
  "refreshToken": ""
}
```

## Error

### Status

```
400 Bad Request
401 Unauthorized
```

### Data

```json
{
  "statusCode": 0,
  "message": [""],
  "error": ""
}
```
